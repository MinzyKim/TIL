# R 1일차

> - R은 객체지향 프로그래밍 언어 - 데이터, 함수, 차트등 모든 데이터는 객체 형태로 관리
> - R은  통계 분석과 data의 시각화를 소프트웨어 환경
> - R은  데이터분석에 필요한 최신 알고리즘, 방법론등의 패키지의 집합이다
> - R은  data의 시각화를 위한 다양한 그래픽 도구를 제공
> - R은 모든 객체는 메모리로 로딩되어 고속으로 처리되고 재사용 가능
