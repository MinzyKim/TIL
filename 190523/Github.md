# Github

Github 사용 typora 사용

## 1.typora 문법

각종 마크다운 문법

- `-` 은 bullet point
- #은 대제목
- ##은 중제목
- ###은 소제목



## 2. 마크다운 문법

1.강조 : **강조**

2.기울임 : *기울임*

3.강조,기울임 : ***강조, 기울임***

> 이거는 인용
